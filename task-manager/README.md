# Task Manager App

A simple Task Manager application built using React.  
This project allows users to manage their daily tasks efficiently.

## Features
- Add new tasks
- Edit tasks
- Delete tasks with confirmation
- Search tasks
- Filter tasks
- Dark mode support

## Tech Stack
- React
- JavaScript
- Tailwind CSS
- LocalStorage

## Project Approach
The application is built using React functional components and state management.

Tasks are stored in LocalStorage so that they remain saved even after refreshing the page.

Different components are created to keep the code clean and reusable.

## Learnings
- React component structure
- useState for state management
- useEffect for syncing data with LocalStorage
- Implementing search and filter functionality
- Building reusable UI components

## How to Run the Project

1. Clone the repository
2. Install dependencies

npm install

3. Run the development server

npm run dev
